🤖 钉钉机器人
如何获取 AppKey / AppSecret？
  1. 前往 钉钉开发者后台，在开发者后台，在一键自动创建OpenClaw机器人处点击立即创建。
  2. 平台已自动预设OpenClaw机器人信息，可根据业务需要修改配置后，单击确定进行创建。
  3. 在创建成功后，复制并保存应用的Client ID和Client Secret。
🚀 飞书机器人
如何获取飞书 App ID / App Secret？
  1. 前往 飞书开放平台，创建企业自建应用
  2. 开启「机器人」能力；在「事件订阅」页选择长连接（WebSocket）模式，添加 im.message.receive_v1 事件
  3. 在凭证页复制 App ID 和 App Secret；WebSocket 模式无需公网 IP
🤖 企业微信机器人
如何获取群聊机器人凭据？
  1. 前往 企业微信管理后台 → 管理工具 -> 智能机器人 页面
  2. 点击 创建 智能机器人，选择“API模式”，而非普通模式（默认），或选择已有的群机器人
  3. 在机器人详情页复制 机器人 ID（BotID） 和 机器人 Secret（BotSecret）
  4. 机器人通过 WebSocket 长连接接入，无需公网 IP
💬 企业微信应用
如何配置企业微信回调？
  1. 前往 企业微信管理后台，创建自建应用
  2. 在应用详情页获取 企业ID（CorpID）、AgentID、Secret
  3. 在「接收消息」中设置回调 URL（需公网 IP），填写自定义 Token 和 EncodingAESKey
  4. 回调路径默认为 /wecom/callback，与本服务部署在同一端口

⏰ 定时任务
定时任务使用说明
  • Cron 格式采用标准 5 字段：分 时 日 月 周，时区取全局配置的 TimeZone
  • 每次触发时向数字员工发送 Prompt，完整回复将推送到指定 Webhook
  • Webhook 类型支持 dingtalk（钉钉）、feishu（飞书）、wecom（企业微信群机器人）
  • 消息类型：text = 纯文本，markdown = Markdown 格式（钉钉/企业微信），post = 富文本（飞书）
  • 修改后保存配置即立即热重载，无需重启服务
保存后立即生效： scheduledTasks.*
🔗 Webhook 接口
使用说明
  • 服务会暴露 /webhook/{规则ID} 入站接口，根据规则ID命中对应规则。
  • 入站请求要求 Authorization: Bearer <token>（token 为规则中的 Bearer Token）。
  • 最终发送给大模型的内容 = 预设 Prompt + 本次 webhook POST 的请求内容。
  • 在预设 Prompt 中无需写变量占位符；可直接写“请基于下方 webhook 请求内容进行分析…”。
保存后立即生效： incomingWebhooks.*
